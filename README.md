# 🧠 Brain Stroke Detection System

## 📌 Project Overview

This project focuses on detecting brain stroke from medical images using image processing and machine learning techniques. It includes classification, segmentation, and brain extraction modules.

---

## 📂 Project Structure

```
stroke-detection-project/
│
├── brain_dataset/        # Raw dataset
├── brain_extraction/    # Code to extract brain region
├── classification/     # Stroke vs Normal classification model
├── segmentation/       # Stroke region extraction
├── dataset/            # Processed dataset
```

---

## ⚙️ Features

* Brain region extraction from MRI images
* Stroke vs Normal classification
* Stroke segmentation
* Severity estimation (if implemented)
* Visualization of results

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Scikit-learn / PyTorch (if used)
* Streamlit (for UI)

---

## 🚀 How to Run

### 1. Clone Repository

```
git clone https://github.com/your-username/stroke-detection-project.git
cd stroke-detection-project
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Run Classification

```
python classification/your_model_file.py
```

### 4. Run Segmentation

```
python segmentation/your_segmentation_file.py
```

### 5. Run UI (if Streamlit)

```
streamlit run app.py
```
